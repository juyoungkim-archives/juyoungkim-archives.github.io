GitHub 직접 저장
Personal Access Token (PAT)을 입력하면 data.json을 GitHub에 직접 커밋합니다.
토큰은 이 기기에만 저장되며 서버로 전송되지 않습니다.

토큰 발급 순서:
1. GitHub 로그인 후 우측 상단 프로필 사진 클릭
2. Settings → 좌측 맨 아래 Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic) 클릭
5. Note: 아무 이름 입력 (예: profile-editor)
6. repo 체크박스 선택 → Generate token
7. 생성된 토큰(ghp_로 시작) 복사 → 아래에 붙여넣기
⚠ 토큰은 생성 직후에만 확인 가능합니다. 반드시 복사해 두세요.
